Trabalhando com Listas

Lista Ordenadas

Podemos criar Listas Ordenadas através da tag OL (Ordered List), acompanhado de LI para cada elemento da lista.

  1. Acordar;
  2. Tomar Café;
  3. Escovar od dentes;

Podemos usar Type= e Start= (CSS) para alterar o tipo da lista e por qual valor irá começar.

  1. Almoçar;
  2. Exercitar-se;
  3. Estudar;

Listas Não Ordenadas

Para criar uma lista não ordenada usamos a tag UL (Unordered List), acompanhado também do LI para cada elemento da lista.

  • Vinland Saga;
  • Jujutsu Kaisen;
  • Kimetsu no Yaiba;

Juntando Listas

Somos capazes de misturar as listas.

  1. Ação
    • Dragon Ball
    • Naruto
    • One Punch Man
  2. Aventura
    • Hunter x Hunter
    • Mushoku Tensei
    • Log Horizon
  3. Suspence
    • Attack on Titan
    • Death Note
    • Tokyo Ghoul
